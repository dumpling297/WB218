Details:
This is an implementation of a client-side puzzle game using HTML tables and JS.
The puzzle is made up of:
  -n*m table
  -numbers at top and left side of table indicate the number of consecutive
  "marked" blocks in each column (and row). -- unimplemented
  -each cell (block) can be toggled from marked to unmarked
  -when solution is reached, a "congratulations" appears

Other:
  -button to reveal the answer
  -to reset the puzzle
  -to create a new puzzle
  -atleast half the puzzle must have more than 1 consecutive line

board is initialized on start-up.
Dimensions of board must be atleast 12*12
